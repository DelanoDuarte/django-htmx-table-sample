# Django HTMX Boostrap table

## About
Simple example using Django 5 and HTMX to load table with pagination.

## Installing

````
pip install -r requirements.txt
````
## Executing

### Migrations (Optional)

- Build
````
./manage.py makemigrations
````

- Execute
````
./manage.py migrate
````

## Start

```
./manage.py runserver 
```
